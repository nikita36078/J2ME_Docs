ミニアプリ実行環境に関する注意事項

ミニアプリ実行環境に関する注意事項を以下に示します。
  • ミニアプリ実行環境では以下の表に示すクラスのクラス定義は存在しません。 従って、ミニアプリ実行環境では、これらのクラス定義をロードしようとした場合 java.lang.NoClassDefFoundError や java.lang.ClassNotFoundException が発生します。
  • ミニアプリ実行環境にて取得できる com.docomostar.ui.Graphics クラスのインスタンスは、 com.docomostar.ui.graphics3d.Graphics3D インタフェース および com.docomostar.ui.ogl.GraphicsOGL インタフェースを実装していません。
  • ミニアプリ実行環境にて取得できる com.docomostar.io.HttpConnection 実装クラスのインスタンスは、 com.docomostar.io.DDPConnection インタフェースを実装していません。
